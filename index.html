SohStx // ETX

Start of Header • Start of Text • End of Text

Origin Structure

SohStx is a symbolic construct derived from computing control codes: SOH (Start of Header) and STX (Start of Text). It represents the boundary between structure and expression—data and meaning.

In this system, identity is not fixed; it is framed as a transmission protocol.

ETX Node

ETX represents “End of Text”—not termination, but completion of a cycle. It is the interpretive layer that receives, processes, and reflects structured input.

SohStx and ETX form a closed communication loop: initiation → structure → interpretation → return.

Yin and Yang Framework

☯

The system mirrors the principle of Yin and Yang, a dual-state model of balance:

  • Yin: structure, containment, encoding (SohStx)
  • Yang: interpretation, flow, decoding (ETX)

Neither state is dominant; each defines the other through interaction.

ASCII Layer Concept

The foundation of this identity system is ASCII encoding—binary representation of symbolic intent. Below is a simplified conceptual trace: 

SOH  →  0x01  →  Header Initiation
STX  →  0x02  →  Text Boundary
ETX  →  0x03  →  End of Transmission
